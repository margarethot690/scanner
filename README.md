## 🔧 How to Use scanner

scanner displays updated blockchain data in an easy-to-understand way. Here are some key parts of the app:

### Real-Time Block Feed

The main screen shows new blocks as they appear on DecentralChain. You can see:

- Block height and timestamp
- Number of transactions
- Miner information
- Block size and gas used

This feed updates every few seconds without refreshing.

### Transaction Tracing

Search for a specific transaction ID (TXID) to see details:

- Sender and recipient addresses
- Amounts transferred
- Fees paid
- Status (confirmed or pending)

You can follow how funds move across multiple addresses with ease.

### Address Analytics

Enter any address to check:

- Balance and transaction count
- Recent activity timeline
- Linked DEX pairs and token holdings

This helps understand user or contract activity.

### DEX Pairs and Market Data

scanner lists available DecentralChain decentralized exchange pairs. View:

- Current prices and volume
- Liquidity pool sizes
- Recent trades and trends

This helps track decentralized finance activity on the network.

### Network Maps

Visual maps show connections between nodes and blocks. Use this to explore network health and structure.

### Node Management

If you run a node, scanner helps you manage it. Features include:

- Node status and uptime
- Peer connections
- Performance statistics

Use this section to keep your node running smoothly.
